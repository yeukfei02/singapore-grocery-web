# singapore-grocery-web

singapore-grocery-web

url: <https://singapore-grocery-web.vercel.app/>

## Requirement

- install yarn
- install node (v14+)

## Testing and run

```zsh
$ yarn

// development
$ yarn run dev

// build dist folder
$ yarn run build

// preview
$ yarn run preview

// lint code
$ yarn run lint

// format code
$ yarn run format
```

open localhost:5000
